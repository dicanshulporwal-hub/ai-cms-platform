# Sample Templates

This folder contains sample HTML templates that can be imported into the CMS using the HTML Template Importer.

## Azure Bay Beach Resort

A luxury beach resort website template featuring:
- Fixed navigation with glass-morphism effect
- Full-screen hero section with gradient background and stats
- Room/suite cards with pricing and features
- Amenities grid with hover effects
- Restaurant/dining cards
- Photo gallery with masonry-style grid
- Guest testimonials with star ratings
- Contact section with booking form
- Newsletter subscription in footer
- Fully responsive design (mobile/tablet/desktop)

### How to Import

**Method 1: Upload ZIP**
1. Go to Admin → Templates → Import HTML → Upload ZIP
2. Select `azure-bay-beach-resort.zip`
3. Fill in license info (MIT license, source: this repo)
4. Click "Upload & Analyze"
5. Review detected regions and save as draft

**Method 2: Paste HTML/CSS**
1. Go to Admin → Templates → Import HTML → Paste HTML/CSS
2. Copy contents of `beach-resort/index.html` into HTML field
3. Copy contents of `beach-resort/styles.css` into CSS field
4. Fill in license info
5. Click "Analyze & Convert"
6. Review and save as draft
